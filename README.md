O **NETDA Quiz** é uma aplicação interativa e visualmente moderna desenvolvida para orientar futuros estudantes da **Escola de Tecnologias Digitais Aplicadas (Sintra)** do **Iscte-IUL**. Através de um questionário dinâmico de 10 perguntas, a aplicação analisa o perfil do utilizador e recomenda a licenciatura que melhor se adapta aos seus interesses e objetivos de carreira.

---

## ✨ Funcionalidades

- 🧠 **Questionário Inteligente:** 10 perguntas focadas em interesses, competências e visão de futuro.
- 📊 **Cálculo de Afinidade:** Sistema de pontuação ponderado que associa respostas a 10 licenciaturas diferentes.
- 🎨 **Interface Premium:** Design moderno com animações suaves, transições de estado e suporte visual para cada curso.
- 🏢 **Identidade dos Núcleos:** Destaque para os núcleos de estudantes **NEST** e **NETDA**, com cores e logos personalizados.
- 📱 **Totalmente Responsivo:** Experiência otimizada para Desktop, Tablet e Mobile.

## 📚 Cursos Abrangidos

A aplicação recomenda cursos divididos entre os dois núcleos principais:

### **NEST (Socioeconómicas e Tecnologias)**
- Política, Economia e Sociedade (PES)
- Tecnologias Digitais e Gestão (TDG)

### **NETDA (Tecnologias Digitais e Aplicadas)**
- Desenvolvimento de Software e Aplicações (DSA)
- Matemática Aplicada e Tecnologias Digitais (MATD)
- Tecnologias Digitais e Automação (TDA)
- Tecnologias Digitais, Edifícios e Construção Sustentável (TDECS)
- Tecnologias Digitais Educativas (TDE)
- Tecnologias Digitais e Inteligência Artificial (TDIA)
- Tecnologias Digitais e Saúde (TDS)
- Tecnologias Digitais e Segurança de Informação (TDSI)

## 🛠️ Tecnologias Utilizadas

- **React 19** - Biblioteca core para a interface.
- **Vite** - Build tool ultrarrápida.
- **Lucide React** - Conjunto de ícones minimalistas.
- **Vanilla CSS** - Estilização personalizada com foco em performance e design moderno.

## 🚀 Como Executar Localmente

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/NETDA-Iscte/NETDA-Quiz.git
   ```

2. **Instalar dependências:**
   ```bash
   npm install
   ```

3. **Iniciar o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Aceder no browser:**
   Abre o link indicado no terminal (geralmente `http://localhost:5173`).
